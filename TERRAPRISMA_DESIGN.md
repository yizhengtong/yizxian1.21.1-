# 泰拉棱镜（Terraprisma）

> 来源：Terraria 1.4 "旅途的终点" — 光之女皇白天无伤击败 100% 掉落  
> 目标：在 Yiz Xian Mod（Minecraft NeoForge 1.21.1）中复刻

---

## 1. 物品面板

| 属性 | 值 | 说明 |
|---|---|---|
| **注册名** | `terraprisma_scroll` | 游戏内 ID |
| **中文名** | 泰拉棱镜卷轴 | |
| **英文名** | Terraprisma Scroll | |
| **物品类型** | 武器卷轴 | 实现 `IWeaponItem` |
| **堆叠上限** | 1 | 不可堆叠 |
| **耐久度** | 无 | 不存在耐久设定 |
| **稀有度** | `EPIC`（紫色） | |

---

## 2. 使用方式

| 操作 | 行为 |
|---|---|
| **右键**（空手/对空气） | 召唤 1 柄泰拉棱镜浮游剑 |
| **Shift + 右键** | 移除 1 柄泰拉棱镜浮游剑（最近召唤的） |
| **已达上限时右键** | 无效果（不消耗不报错） |
| **无剑时 Shift+右键** | 无效果 |

- **召唤条件**：当前玩家拥有的浮游剑数量 < 8
- **消耗**：无（不消耗物品，不消耗耐久）
- **冷却**：无（想召就召，仅受上限约束）

---

## 3. 浮游剑实体（TerraprismaEntity）

### 3.1 基础面板

| 属性 | 值 |
|---|---|
| **实体 ID** | `yizxianmod:terraprisma` |
| **每柄伤害** | `9`（4.5❤） |
| **伤害类型** | 通用伤害（受护甲 / 保护附魔 / 抗性减免） |
| **攻击间隔** | `0.4 秒`（8 ticks） |
| **同时存在上限** | `8` 柄 |
| **持续时间** | **无限**（直到玩家死亡 / 离线 / 手动移除） |
| **可否被攻击** | 不可破坏（永生实体，不受任何伤害） |
| **碰撞体积** | 无（不阻挡玩家和生物移动） |

### 3.2 无敌帧破解

| 特性 | 说明 |
|---|---|
| **机制** | 攻击前将目标的 `invulnerableTime` 临时设为 0，攻击后自然恢复为 10 ticks |
| **效果** | 泰拉棱镜的攻击**必定命中**，不受目标当前无敌帧影响 |
| **对其他伤害源的影响** | 无——攻击后目标 `invulnerableTime` = 10，其他武器仍需等待无敌帧结束才能命中 |
| **设计意图** | 8 柄剑各自独立攻击，不互相"吃掉"伤害判定；但不会让其他武器也绕过无敌帧 |

### 3.3 行为状态机

```
                 ┌──────────────────────────┐
                 │      SWIRL（环绕）         │
                 │  以玩家为中心 2.5 格轨道    │
                 │  角速度 ~2.5 rad/s        │
                 │  Y 轴正弦浮动 ±0.5 格      │
                 └──────────┬───────────────┘
                            │ 检测到敌对生物在 8 格内
                            │ 且攻击冷却 = 0
                            ▼
                 ┌──────────────────────────┐
                 │      DASH（冲刺）          │
                 │  直线飞向目标 0.8 block/tick│
                 │  接触 → 造成 9 伤害         │
                 │  触发 8 tick 冷却          │
                 └──────────┬───────────────┘
                            │ 命中 / 目标死亡 / 超过 3 秒未命中
                            ▼
                     返回 SWIRL（环绕）
```

### 3.4 多剑相位分布

8 柄剑在环绕轨道上均匀分布初始相位角：

| 第 N 柄 | 初始相位 |
|---|---|
| 1 | 0° |
| 2 | 45° |
| 3 | 90° |
| 4 | 135° |
| 5 | 180° |
| 6 | 225° |
| 7 | 270° |
| 8 | 315° |

> 新召唤的剑取下一个空闲相位，移除剑后相位重新分配。

### 3.5 存活逻辑

| 条件 | 行为 |
|---|---|
| Owner 死亡 | 所有浮游剑立即 despawn |
| Owner 离线 | 所有浮游剑立即 despawn |
| Owner 切换维度 | 浮游剑跟随传送 |
| 浮游剑距 Owner > 32 格 | 立即传送到 Owner 身边 |
| 服务器重启 | 浮游剑不持久化（消失） |

## 4. 伤害计算示例

| 场景 | 理论 DPS | 说明 |
|---|---|---|
| 1 柄剑 | 22.5 | 9 ÷ 0.4 |
| 4 柄剑 | 90 | 4 × 22.5 |
| 8 柄剑（满） | **180** | 8 × 22.5 |

> 注意：实际 DPS 受目标护甲、剑的索敌效率、目标移动等因素影响，达不到理论值。

**对比原版天花板：**

| 武器 | 理论 DPS |
|---|---|
| 下界合金剑（普攻） | ~12.8 |
| 下界合金斧（普攻） | ~10 |
| **泰拉棱镜 8 柄** | **~180** |
